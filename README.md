# Elang
Embedded Language

## Quick Start
Copy [elang.h](elang.h) into your project and use it like this:

```c
#define ELANG_IMPLEMENTATION
#include "elang.h"
```

## Example
See [test.c](test.c)

```console
$ ./build.sh
$ ./test example
```
